# DOM Event Challenge

A beginner JavaScript project demonstrating DOM manipulation and event handling.

## 🚀 Features

- 📌 Click counter
- 🎨 Toggle background color
- 🔔 Alert button
- ✏️ Dynamically updates text using JavaScript
- ⚡ Uses Event Listeners for user interaction

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

```
dom-event-challenge/
├── index.html
├── README.md
```

## 🎯 What I Learned

- Selecting elements using `getElementById()`
- Using `addEventListener()`
- Updating page content with `innerText`
- Changing CSS styles using JavaScript
- Creating a click counter
- Implementing a background color toggle
- Writing simple interactive web applications
